# Dexmon v2 - Pokémon Card App

This project is a **Pokémon card UI component** built with **Next.js** and **TypeScript**. It dynamically displays a Pokémon's information, such as name, stats, abilities, and types, with enhanced visual styling.

---

## Features
- **Responsive Pokémon Card Design**
  - Displays Pokémon name, HP, sprite, types, and stats.
- **Dynamic Backgrounds**
  - Type-based gradient backgrounds for visual appeal. (uses a hex color mixer to create a unique background color.)
- **Scrollable Ability Description**
  - Shows the Pokémon's ability with a scrollable description area. (I find this more reliable than making the text smaller or changing the card's aspect ratio.)
- **Type Icons**
  - Icons for each Pokémon type alongside their names. (The contrast is not the best with the background color for some colors, but I will keep the placement as is.)
- **Interactive Styling**
  - Gradient effects, shadows, and polished UI for an engaging experience. (I tried to make it look like an actual Pokémon card while keeping it visually minimalistic.)

---

## How to Use
1. Clone the repository:
   ```bash
   git clone https://github.com/KubilayKilic/Dexmon.git
     ```
2. Install:
    ```bash
    npm install
    ```
3. Run:
   ```bash
    npm run dev
    Access the app at http://localhost:3000
   ```
